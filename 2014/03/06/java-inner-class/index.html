6、嵌套类

嵌套类，即使用了static修饰的内部类。由于使用了static修饰，因此以下两个问题需要注意： 1、要创建嵌套类的对象，不需要其外围类的对象 2、嵌套类中，不能访问非静态的外部类对象。 普通的内部类，不能有static修饰的变量或者方法，也不能包含嵌套类。但是嵌套类可以包含这些东西。同样的嵌套类中也么有.this指向外部类。

7、接口内部的类

普通情况下，接口内不能放置任何代码，但是嵌套类可以作为接口的一部分。放到接口中的任何类都是pubilc static的

8、为什么使用内部类

每个内部类都能够独立的继承自一个接口的实现，所以无论外围类是否已经继承了某个类或者接口的实现，对于内部类没有影响。
